# Chart Image Placement

Place your visualization images in this folder so they are served by Vite.

Expected filenames:
- plot_1_glucose_histogram.png
- plot_2_bmi_vs_glucose_scatter.png
- plot_3_correlation_heatmap.png

These are referenced in src/data/mockData.js as:
- /charts/plot_1_glucose_histogram.png
- /charts/plot_2_bmi_vs_glucose_scatter.png
- /charts/plot_3_correlation_heatmap.png

You can replace with API URLs later by updating imageUrl values in src/data/mockData.js.
